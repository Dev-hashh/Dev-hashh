> `Python` `FastAPI` `FAISS` `Sentence-Transformers` `Groq API`

End-to-end Retrieval-Augmented Generation pipeline — ingest a PDF, get context-grounded answers back. Deployed live with Groq's LPU inference for ultra-low latency.

### 🕵️ [Fake Review Detection](https://github.com/Dev-hashh/Fake-Review-Detection) — NLP Classifier
> `Python` `Scikit-learn` `TF-IDF` `Logistic Regression` `Pandas`

ML pipeline that classifies fake vs. genuine reviews across 50,000 samples — **~96% accuracy, F1 ~0.92**.

- TF-IDF (unigrams + bigrams, 5K features) + behavioral features via scipy sparse matrices
- Feature ablation confirmed the model learns purely from text — no star-rating cheating
- Model interpretability via LR coefficients; serialized with joblib

[![Repo](https://img.shields.io/badge/GitHub-Repo-181717?style=flat-square&logo=github)](https://github.com/Dev-hashh/Fake-Review-Detection)

> `MongoDB` `Express.js` `React` `Node.js` `Cloudinary` `Stripe`

Full-stack property listing and booking platform with auth, payments, and image uploads.

- Session-based auth with Passport.js + connect-mongo
